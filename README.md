# Boulangerie Luna

架空のパン屋「Boulangerie Luna」のランディングページです。

## 概要

Next.js と Tailwind CSS を使って制作した、パン屋をテーマにした静的サイトです。実在する店舗ではなく、コーディング練習を目的として制作しました。デザインは適当です。

## 使用技術

- **Next.js** (App Router)
- **Tailwind CSS**
- **TypeScript**

## 実装内容

- フルスクリーンのヒーロー画像ヘッダー
- スクロールリンク付きナビゲーション
- CONCEPT / MENU / ACCESS / CONTACT の各セクション
- Google Maps 埋め込み
- お問い合わせフォーム
- フッター
